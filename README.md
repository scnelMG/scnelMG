## About

데이터 분석과 머신러닝 프로젝트를 포트폴리오로 정리하고 있습니다.

모델을 학습시키는 것에서 끝내지 않고, **문제 정의, 데이터 공개 경계, 모델링 의사결정, 결과 근거, 재현 가능성, 한계**가 함께 보이는 저장소를 지향합니다.

관심 분야는 **Tabular ML, 추천/랭킹, NLP, Vision/VLM, 금융/보험 데이터, 공공데이터 서비스**입니다.

## Portfolio Principles

- **문제 정의**: 어떤 의사결정을 돕는 프로젝트인지 먼저 설명합니다.
- **역할 분리**: 팀 프로젝트는 단독 제작처럼 보이지 않게 기여 범위를 구분합니다.
- **기술적 의사결정**: 모델, 검증, feature engineering 선택 이유를 남깁니다.
- **결과 근거**: 단일 점수보다 pipeline, validation, feature evidence를 함께 제시합니다.
- **공개 안전성**: 원본 데이터, 비공개 자료, 대용량 artifact, 개인정보 가능 자료를 제외하고 문서화합니다.
