## 문서화 기준

각 저장소는 단순 결과 나열보다 아래 기준으로 정리했습니다.

- **문제 정의**: 어떤 도메인 문제를 풀었는지 먼저 설명
- **내 역할**: 팀 프로젝트에서 맡은 판단과 구현 범위 분리
- **기술적 의사결정**: 모델·데이터·검증 방식을 왜 선택했는지 기록
- **재현 가능성**: 실행 방법, 입력 데이터 조건, 한계 명시
- **공개 안전성**: 원본 데이터, 개인정보 가능 자료, 대용량 산출물, 대회 비공개 자료를 구분
